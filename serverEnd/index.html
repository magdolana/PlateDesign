Download LS-SEED

Here you can download LS-SEED for your OS. There's a few things you should know before proceeding:

Windows

It's recommended that you don't place the contents of the downloaded archive in the "Program Files" or "Program Files (x86)" folders. If you place it here, the auto updater will fail unless you grant yourself full write and execute access on the folder where you put Ls-Seed.

OS X

Your first attempt at opening the application will be prevented unless you have set "Allow applications downloaded from:" to "Anywhere" in the General Security & Privacy settings panel. If you are having trouble with this, you can right click on "ls-seed.app", select open, then open again, and the problem should be solved. This problem will be fixed before the end of the beta.

Make sure you have write and execute access to "ls-seed.app", or the auto updater will not function properly.

Linux

Make sure you have execute and write permissions on all the files within your ls-seed folder. If you don't set the required permissions, the auto updater won't function correctly.
